Layout

Background Image:

The background image is an artistic time vortex that serves as the visual foundation of the artwork. The image has been visually processed to enhance the thematic atmosphere.

View Box:

A torn old newspaper effect overlays the background, forming a foreground. The torn portion reveals the background image, creating layers and visual impact.

Centered Text:

A poetic phrase displayed in the center of the screen, reinforcing the theme of the artwork. The text stands out clearly against the background.

Design System Label:

Located in the top-right corner, the label invites users to click and view the Design System information.

Popup Panel:

Clicking the Design System label triggers a sliding white popup panel from the right, containing detailed design information.

Interaction

Centered Text Interaction:

When the centered text is clicked, the screen shakes violently, simulating the distortion of time and space. A lightning effect briefly appears, enhancing the visual dynamic.

Design System Interaction:

Clicking the Design System label slides out the popup from the right. Clicking outside the popup closes it.

Popup Scroll Support:

If the popup content exceeds the visible area, users can scroll vertically to view all information, improving the user experience.

Loading and Dynamic Effects:

The background and view box automatically adjust to the screen size upon loading. Smooth animations accompany user interactions, enhancing the immersive experience.
